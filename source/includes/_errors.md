# Errors

The Comms4Docs API uses the following error codes:

| Code | Meaning                                            |
| ---- | -------------------------------------------------- |
| 401  | Unauthorized - You are not logged in as Admin.     |
| 404  | Not Found -- The specified URL could not be found. |
| 429  | Too Many Requests                                  |
| 500  | Internal Server Error                              |
